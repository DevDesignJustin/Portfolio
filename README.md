# kjustin — Personal Portfolio

Personal portfolio site for Justin Kim. Built with Next.js, GSAP, and Tailwind CSS.

## Stack

- **Framework** — Next.js 16 (App Router)
- **Animations** — GSAP + ScrollTrigger
- **Styling** — Tailwind CSS
- **Language** — TypeScript
- **Deployment** — Vercel

## Development

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Project Structure

```
app/
  page.tsx        # Main portfolio page
  data.ts         # All content, theme, and accent config
  layout.tsx      # Root layout and fonts
  globals.css     # Global styles and keyframes
  components/     # Loader, MagneticButton, GitHubStats, etc.
```
